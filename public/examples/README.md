# Imagens de Exemplo

Esta pasta contém imagens de exemplo para testar o sistema.

## Imagens Disponíveis

- `candidato-1.jpg` - Homem de terno (profissional)
- `candidato-2.jpg` - Mulher profissional
- `candidato-3.jpg` - Homem jovem profissional
- `candidato-4.jpg` - Mulher jovem profissional

## Como Usar

1. Acesse o painel admin em `/admin`
2. Vá na aba "Candidatos"
3. Clique em "Adicionar Candidato"
4. No campo de foto, faça upload de uma dessas imagens
5. Preencha os demais campos e salve

## Nota

Estas são imagens genéricas de exemplo do Unsplash (domínio público).
Substitua por fotos reais dos candidatos antes de usar em produção.
